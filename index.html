Web API Mashup



Mashup hyödyntää kahta erilaista sovellusrajapintaa, Google Booksin ja WikiMedian API:a. Lisäksi sovelluksessa käytetään Deck.js -kirjastoa, jonka avulla voidaan nuolinäppäimillä suunnistetaan sivustolla.

Google Books API + Deck.Js

Käyttäjä etsii haluamallaan hakusanallaan kirjoja, joita näytetään 9 kpl per sivu. Kokonaisuudessaan kirjoja haetaan 27 kpl, jotka on jaettu kolmelle eri sivulle tasaisesti. Käyttäjän on mahdollista klikata kirjoja ja päästä suoraan Google Booksin sivuille esim. ostotarkoituksissa.


WikiMedia API

Tämä API hyppää mukaan mukaan ensimmäisen haun jälkeen. Käyttäjän on mahdollista klikata kirjojen tekijöiden nimiä, jolloin WikiMedian kysely ottaa hakuparametrikseen klikatun tekijän. Tieto haetaan suoraan Wikipediasta. Alas laskeutuva modaali näyttää mahdollisen tuloksen käyttäjälle.
